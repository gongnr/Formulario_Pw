Questões Respondidas

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
    • O atributo "placeholder" dado à tag <input> com o atributo 'type="text"' apresenta uma pequena dica que descreve o valor esperado para um campo de entrada, tal como uma amostra do valor ou pequena descrição.
  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
    • O atributo "checked" nas hipóteses acima indicadas determina que um dos campos de entrada deverá estar pré-selecionado quando o documento HTML for carregado.
  3. Na tag <input> para que serve o atributo "value"?
    • O atributo "value" na tag <input> serve para especificar o valor pré-selecionado (padrão) para um campo de entrada.
  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    • O atributo utilizado para agrupar vários botões de rádio em um formulário é o atributo "name". Uma vez que um grupo de botões de rádio é estabelecido, a seleção de um botão daquele grupo automaticamente remove a seleção de outro botão do mesmo grupo.
  5. Para que serve a tag
    • A tag <label> define um rótulo (legenda) para elementos do formulário. Por sua vez, o atributo "for" da referida tag liga tal atributo ao elemento <input> com "id" idêntico.
  6. Na tag <option>, para que serve o atributo "selected"?
    • O atributo selected na tag <option> indica que uma opção está selecionada inicialmente.
  7. Nas tags de formulário, para que serve o atributo "required"?
      8. Em em um elemento de formulário que tenha recebido o atributo "required", seu preenchimento torna-se obrigatório antes da submissão do formulário.
  8. Na tag <form>, para que serve o atributo "action"?
    • O atributo "action" define qual ação será performada quando o formulário é submetido.
  9. Na tag <form>, para que serve o atributo "method"?
      10. O atributo "method" define o método pelo qual os dados do formulário serão transmitidos. Os dados do formulário poderão ser enviados como variáveis do URL (method="get") ou como transação de postagem de HTTP (method="post"). O método padrão é "get".
